## Primary Activities（Porter 价值链）

| # | Activity | CampusShare Implementation |
|---|---|---|
| 1 | **Inbound Logistics** | Onboarding Hosts：采集闲置空间信息（尺寸、地址、可用期），通过 `.edu` 身份认证确保供给质量；Renter 注册并录入存储需求 |
| 2 | **Operations** | 撮合与交易执行：搜索排序、Booking 状态管理、电子协议生成、押金托管、物品照片登记、双方交接时间协调 |
| 3 | **Outbound Logistics** | 物品交接：Renter 与 Host 在校内约定地点完成入库与归还；平台生成交接凭证与时间戳 |
| 4 | **Marketing & Sales** | 校园渠道营销：学生社团合作、宿舍楼公告、国际学生办公室联动、`.edu` 邮件列表推送；暑期前高峰期投放 |
| 5 | **Service** | 售后与信任维护：双向评价、Dispute 仲裁、客户支持、保险理赔协助、数据保留与删除合规 |

## Narrative（嵌入报告时可用）

CampusShare 的 primary activities 紧密围绕 Focused Differentiation 战略：Inbound 与 Marketing 通过 `.edu` 围墙确保双边用户都是认证学生，Operations 通过数据驱动的撮合流程降低交易摩擦，Outbound 聚焦校内步行可达的超短距离交接，Service 通过评价与纠纷机制积累平台信任资产。
